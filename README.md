[link to the project](https://open-bus-map-search.hasadna.org.il/dashboard)

## Contribution:

### Running the project:

- fork the repo
- clone the repo
- `yarn install`
- `yarn start`

### testing the project:
- running the tests - `yarn playwright test`
- running the tests with UI - `yarn playwright test --ui`
- additional helpful flags - https://playwright.dev/docs/test-cli
